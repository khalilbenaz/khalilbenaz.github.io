## ✨ Fonctionnalités

- **Thème jour/nuit automatique** — Bascule automatiquement selon l'heure (6h-18h jour, 18h-6h nuit), avec toggle manuel
- **Particules animées** — Arrière-plan interactif avec particules flottantes
- **Design responsive** — Adapté mobile, tablette et desktop
- **Animations fluides** — Transitions CSS avec cubic-bezier, barres de compétences animées
- **Single Page** — Tout le CV en une seule page performante, zéro dépendance framework

## 🚀 Déploiement

Le site est automatiquement déployé via **GitHub Pages** à chaque push sur `main`.

Pour lancer en local :

```bash
git clone https://github.com/khalilbenaz/khalilbenaz.github.io.git
cd khalilbenaz.github.io
# Ouvrir index.html dans un navigateur, ou :
python3 -m http.server 8000
# → http://localhost:8000
```
